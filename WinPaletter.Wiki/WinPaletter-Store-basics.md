## 6. Themes aspects review

- To prevent accidental actions, a window with all edited aspects in this theme will be listed

![alt text](https://github.com/Abdelrhman-AK/WinPaletter/blob/master/WinPaletter.Wiki/Asset/Store/6_0.png?raw=true)

- Pressing on this will proceed theme applying with all selections you made

![alt text](https://github.com/Abdelrhman-AK/WinPaletter/blob/master/WinPaletter.Wiki/Asset/Store/6_1.png?raw=true)

- Pressing on this will proceed theme applying with all selections are checked whatever you have chosen

![alt text](https://github.com/Abdelrhman-AK/WinPaletter/blob/master/WinPaletter.Wiki/Asset/Store/6_2.png?raw=true)

## 7. Resources pack download

- If this theme has a resources pack, this window will appear with information about this download process

- After it completes, theme will be applied automatically

- Resources pack includes files (images (wallpapers), sounds and screensaver) that were stored in author's devices and these files are mainly not on your device, to help WinPaletter apply the theme correctly

![alt text](https://github.com/Abdelrhman-AK/WinPaletter/blob/master/WinPaletter.Wiki/Asset/Store/7.png?raw=true)
