## 1. Enable/Disable lock toggle

Make it checked to make WinPaletter apply changes to Screen Saver

![alt text](https://github.com/Abdelrhman-AK/WinPaletter/blob/master/WinPaletter.Wiki/Asset//ScreenSaver/0.png?raw=true)

***

## 2. `Optional` Toolbar buttons

### 1. Open from WinPaletter theme
This will copy preferences from a WinPaletter theme file into current open theme

### 2. Open from Classic .theme file
This will copy preferences from current classic Windows theme file (.theme) into current open theme (looks like an undo)

### 3. Open from Current applied one
This will copy preferences from current Screen Saver settings into current open theme (looks like an undo)

### 4. Open from Default Windows
This will copy preferences from default Windows theme into current open theme

![alt text](https://github.com/Abdelrhman-AK/WinPaletter/blob/master/WinPaletter.Wiki/Asset//ScreenSaver/1.png?raw=true)

## 3. Parts

![alt text](https://github.com/Abdelrhman-AK/WinPaletter/blob/master/WinPaletter.Wiki/Asset//ScreenSaver/2.png?raw=true)

- You can select Screen Saver file, there are some files by default in `%windir%/system32`

- You can change interval that will trigger it (timeout) in seconds

- You can logoff after closing Screen Saver and protect your Windows by your user password (on resume, password protect)

![alt text](https://github.com/Abdelrhman-AK/WinPaletter/blob/master/WinPaletter.Wiki/Asset//ScreenSaver/3.png?raw=true)

1. If there a selected screen saver file, it will play it in full screen

2. If there a selected screen saver file, it will play it in preview area

3. If there a selected screen saver file and it is open, it will stop it

4. If there a selected screen saver file, it will open its settings

- **Important tip:** there may be a lag after playing a screen saver file (according to your device resources/performance)

***

## 4. Actions

1. **Load into current theme:** it will load current Screen Saver preferences into current open WinPaletter theme, and will close this form so you can continue modifying other Windows aspects and finally press on Apply on main form to apply all aspects in the open theme.

2. **Quick apply:** it will apply current Screen Saver preferences instantly to test their real effects on system.

3. **Cancel:** it will close this form without loading any modification made to Screen Saver
