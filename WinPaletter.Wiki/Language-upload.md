1. Create a GitHub Account if you don't have an account, or login if you already have one.

2. Visit [WinPaletter Repository](https://github.com/Abdelrhman-AK/WinPaletter)

3. Fork (You will find this button in the upper part of the page).

4. Now visit your personal account main page, you will find WinPaletter Forked (as if cloned to your account if you don't know the meaning of Fork).

5. Navigate to directory `Languages` and create a new file or edit it if you already uploaded a previous one.

6. After creating the file, open it and press on edit (pencil) icon.

7. Override contents of the created file by the contents of file you previously created.

8. Confirm saving the file.

9. Create a pull request, once this request is approved, your new language file will be merged with the Main WinPaletter Repository and so anyone can access your translation\language from [this link](https://github.com/Abdelrhman-AK/WinPaletter/tree/master/Languages)
