## 2. `Optional` Toolbar buttons

### 1. Open from WinPaletter theme
This will copy preferences from a WinPaletter theme file into current open theme

### 2. Open from Classic .theme file
This will copy preferences from current classic Windows theme file (.theme) into current open theme (looks like an undo)

### 3. Open from Current applied one
This will copy preferences from current sounds settings into current open theme (looks like an undo)

### 4. Open from Default Windows
This will copy preferences from default Windows theme into current open theme

### 2. Categories

They are: `main events`, `dialogs`, `window & menus`, `devices`, `fax`, `explorer events`, `notifications`, `alarms`, `ringtones`, `speech recognition`, `NetMeeting` and `others`

![alt text](https://github.com/Abdelrhman-AK/WinPaletter/blob/master/WinPaletter.wiki/Asset//Sounds/3.png?raw=true)

#### 1. Main events

- This category includes events of: Startup (imageres.dll), Startup (registry), Shutdown, Logoff, Logon, Unlock and Theme change.

- Shutdown, Logoff and Logon sounds in Windows 8 and later are changed by creating tasks in Windows Task Scheduler, so your
experience with these sounds won't be perfect (Unlike Windows 7 and below).

#### 2. Dialogs

- This category includes events of: Information, Question, Exclamation, Asterisk, Critical stop, UAC and General protection fault

#### 3. Window & menus

- This category includes events of: Open, Close, Maximize, Minimize, Restore down, Restore up, Menu popup and Menu click

#### 4. Devices

- This category includes events of: Device connection, Device disconnection, Device failure, Low battery, Critical battery alarm, Print completed and Proximity connection

#### 5. Fax

- This category includes events of: Line rings, Beep, New, Sent and Error

#### 6. Explorer events

- This category includes events of: Folders navigation, Empty Recycle Bin, Move menu item, Activating document, Show band, Security band, Blocked popup, Feed discovered and Search provider discovered

#### 7. Notifications

- This category includes events of: Default beep, Notification, Internet message, Message nudge, Mail notification, Mail beep, Proximity, Reminder and SMS

#### 8. Alarms

- This category includes events of default Windows alarms from 1 to 10

#### 9. Ringtones

- This category includes events of default Windows ringtones from 1 to 10

#### 10. Speech recognition

- This category includes events of: Disambiguation numbers, Disambiguation panels, Misrecognition, On, Off and Sleep

#### 11. NetMeeting

- This application is installed by defaults in Windows XP and it has sounds entries.

- This category includes events of: Person joins, Person leaves, Receive call and Receive request to join

#### 12. Others

- This category includes events of: CC select

***

## 4. Actions

1. **Load into current theme:** it will load current sounds preferences into current open WinPaletter theme, and will close this form so you can continue modifying other Windows aspects and finally press on Apply on main form to apply all aspects in the open theme.

2. **Quick apply:** it will apply current sounds preferences instantly to test their real effects on system.

3. **Cancel:** it will close this form without loading any modification made to sounds
