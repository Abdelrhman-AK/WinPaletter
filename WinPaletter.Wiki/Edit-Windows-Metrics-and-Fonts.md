## 2. `Optional` Toolbar buttons

### 1. Open from WinPaletter theme
This will copy preferences from a WinPaletter theme file into current open theme

### 2. Open from Visual styles file
This will copy preferences from a Windows theme file (.theme or .msstyles) into current open theme

### 3. Open from Current applied one
This will copy preferences from current metrics and fonts settings into current open theme (looks like an undo)

### 4. Open from Default Windows
This will copy preferences from default Windows theme into current open theme

![alt text](https://github.com/Abdelrhman-AK/WinPaletter/blob/master/WinPaletter.Wiki/Asset/MetricsFonts/1.png?raw=true)

## 4. Actions

1. **Load into current theme:** it will load current metrics and fonts preferences into current open WinPaletter theme, and will close this form so you can continue modifying other Windows aspects and finally press on Apply on main form to apply all aspects in the open theme.

2. **Quick apply:** it will apply current metrics and fonts preferences instantly to test their real effects on system.

3. **Cancel:** it will close this form without loading any modification made to metrics and fonts

![alt text](https://github.com/Abdelrhman-AK/WinPaletter/blob/master/WinPaletter.Wiki/Asset/MetricsFonts/8.png?raw=true)
