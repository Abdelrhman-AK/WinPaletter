## 3. Parts
There are main three areas in which you can customize these aspects

![alt text](https://github.com/Abdelrhman-AK/WinPaletter/blob/master/WinPaletter.wiki/Asset/WindowsColors/mainformparts.png?raw=true)

### 1. **Toggles**
You can change some aspects by these toggles; including Windows dark/light mode, transparency, apply accent on elements like taskbar and start menu

You can change titlebars colors; including active and inactive colors with a toggle to disable\enable titlebar colorization

### 3. **Accents** 
You can change different Windows colors, and these colors are illustrated by labels

1. **Colorization blur balance:** changes blur amount controlling colorization color (in another way, change intensity of transparency/aero glass)

# Windows XP

## 1. Target OS Preview 
Change it to Windows XP

![alt text](https://github.com/Abdelrhman-AK/WinPaletter/blob/master/WinPaletter.wiki/Asset/OSPreview/XP.png?raw=true)

## 2. Parts
There are two main areas in which you can customize Windows XP themes

![alt text](https://github.com/Abdelrhman-AK/WinPaletter/blob/master/WinPaletter.wiki/Asset/WindowsColors/mainformparts_xp.png?raw=true)

### 1. **Theme** 
You can change Windows theme (Luna blue, Luna Green, Luna Silver, external visual styles and Windows classic)

### 2. **External theme** 
You can load an external theme provided that:
- `External` in theme section is selected
- `UX-Theme` patch for Windows is done

## 3. **Overriding aspects** 

Theme/Visual styles has values for classic colors, metrics and fonts inside their files. When they are applied, these internal values are applied too.

- If `Colors in classic colors` is not checked, WinPaletter will apply classic colors from the loaded values but not from the loaded Theme/Visual styles, and vice versa.

- If `Metrics values` is not checked, WinPaletter will apply metrics from the loaded values but not from the loaded Theme/Visual styles, and vice versa.

- If `System fonts` is not checked, WinPaletter will apply fonts from the loaded values but not from the loaded Theme/Visual styles, and vice versa.
