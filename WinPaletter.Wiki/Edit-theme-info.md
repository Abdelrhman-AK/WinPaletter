Edit info form is composed of 3 main tabs

![alt text](https://github.com/Abdelrhman-AK/WinPaletter/blob/master/WinPaletter.Wiki/Assets//ThemeInfo/i0.png?raw=true)

## 1. Main info
Change theme name, its version and type its description

![alt text](https://github.com/Abdelrhman-AK/WinPaletter/blob/master/WinPaletter.Wiki/Assets//ThemeInfo/i1.png?raw=true)

> If you are willing to upload your theme to WinPaletter Store, type tags inside description to make searching it easier

Type your name and you social media link `link is optional`.
Author name is your Windows account name by default.

![alt text](https://github.com/Abdelrhman-AK/WinPaletter/blob/master/WinPaletter.Wiki/Assets//ThemeInfo/i2.png?raw=true)

## 2. Theme resources pack

This is useful for creating a pack containing files used in your theme (images, sounds, screensavers, ...) and hence, it will be better for sharing and for WinPaletter Store to download the associated resources

- If you want to export a pack for current open theme, check this choice:

![alt text](https://github.com/Abdelrhman-AK/WinPaletter/blob/master/WinPaletter.Wiki/Assets//ThemeInfo/i3.png?raw=true)

- If you checked the option before, type in this text box the credits/licenses of images, sounds, screensavers, ... used in your theme

![alt text](https://github.com/Abdelrhman-AK/WinPaletter/blob/master/WinPaletter.Wiki/Assets//ThemeInfo/i4.png?raw=true)

## 3. `Optional` Store info

Open it if you are going to upload this theme to WinPaletter Store

- Change the two descriptive colors (They should give users an idea about your theme)

![alt text](https://github.com/Abdelrhman-AK/WinPaletter/blob/master/WinPaletter.Wiki/Assets//ThemeInfo/i5.png?raw=true)

- Slide this track bar to change background decoration (This will give your theme its look inside WinPaletter Store)

![alt text](https://github.com/Abdelrhman-AK/WinPaletter/blob/master/WinPaletter.Wiki/Assets//ThemeInfo/i6.png?raw=true)

- Check operation systems on which your theme will work (or designed for it)

![alt text](https://github.com/Abdelrhman-AK/WinPaletter/blob/master/WinPaletter.Wiki/Assets//ThemeInfo/i7.png?raw=true)

## 4. Finish

- Press on Load into current theme

![alt text](https://github.com/Abdelrhman-AK/WinPaletter/blob/master/WinPaletter.Wiki/Assets//ThemeInfo/i8.png?raw=true)
