## 1. Disable ExplorerPatcher synchronization

- By default, synchronization of Windows 11 preview with current ExplorerPatcher settings if this application is already installed is enabled.

- To disable this synchronization: Open Settings `>` ExplorerPatcher `>` Uncheck `Synchronize with Windows 11 preview if installed`

![alt text](https://github.com/Abdelrhman-AK/WinPaletter/blob/master/WinPaletter.wiki/Asset/Settings/EPNoSync.png?raw=true)

***

## 2. Force (Deflect) ExplorerPatcher synchronization

- You even can force this synchronization even if ExplorerPatcher is not installed (for testing purposes)

-  Open Settings `>` ExplorerPatcher `>` Check `Deflect (Force the preview' of Windows 11 to simulate as if ExplorerPatcher is installed even if you are not using Windows 11)`

- You can also deflect applied ExplorerPatcher settings

![alt text](https://github.com/Abdelrhman-AK/WinPaletter/blob/master/WinPaletter.wiki/Asset/Settings/EPDeflect.png?raw=true)
