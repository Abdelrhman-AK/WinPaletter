# Running WinPaletter on Windows XP

## Steps:

1. Download the latest release of [Skulltrail192/One-Core-API-Binaries](https://github.com/Skulltrail192/One-Core-API-Binaries).  
2. Install it on the target real or virtual machine.  
3. Restart Windows XP.  
4. Follow the documentation to install **.NET Framework 4.8** on Windows XP:  
   [Build .NET Framework 4.8 for legacy operating systems (Windows XP, Vista)](https://github.com/Abdelrhman-AK/WinPaletter/wiki/Build-.NET-Framework-4.8-for-legacy-operation-systems-(Windows-XP,-Vista))  
5. Restart Windows XP again.  
6. Read the important notes below.  
7. Try launching WinPaletter.

> [!Important]
> 1. WinPaletter is **not stable on Windows XP**; test it first in a virtual machine.  
> 2. You may encounter unexplained crashes on startup, during use, or when closing the application.  
> 3. If the classic theme is enabled, theme previews may not work due to limitations in Visual Styles Previewer. Apply a different theme first, then reopen WinPaletter.
