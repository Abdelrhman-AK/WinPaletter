## 2. Lock/Unlock toggle

Make it checked to make WinPaletter can modify LogonUI screen

![alt text](https://github.com/Abdelrhman-AK/WinPaletter/blob/master/WinPaletter.Wiki/Asset/LogonUI/logonui_81_0.png?raw=true)

## 3. Contents

There are 2 main parts:

![alt text](https://github.com/Abdelrhman-AK/WinPaletter/blob/master/WinPaletter.Wiki/Asset/LogonUI/logonui_81_1.png?raw=true)

### 1. Source

You can choose a source from the following:

1. **System default:** note that Windows 8.1 has a button that makes you choose a stock Windows 8.1 lock screen image, while there is no button for Windows 7

2. **Current wallpaper:** it will make current wallpaper as a background for LogonUI screen

3. **Solid color:** it will make the selected color as a background for LogonUI screen

4. **Custom image:** it will make the selected image as a background for LogonUI screen

### 2. Effects

You can apply effects to the selected source

1. **Gray-scale**

2. **Blurred:** with a track bar to control its intensity.

3. **Noise:** it has two types; Windows 11/10 acrylic noise and aero glass reflection. It has also a track bar to control its intensity.
