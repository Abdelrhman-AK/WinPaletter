## 1. Start language creation

- Open Settings > Language > Language development tools

![alt text](https://github.com/Abdelrhman-AK/WinPaletter/blob/master/Media/Wiki/LanguageCreation/0.png?raw=true)

## 2. Open GUI language editor

![alt text](https://github.com/Abdelrhman-AK/WinPaletter/blob/master/Media/Wiki/LanguageCreation/1.png?raw=true)

## 3. Toolbar

![alt text](https://github.com/Abdelrhman-AK/WinPaletter/blob/master/Media/Wiki/LanguageCreation/2.png?raw=true)

- **Open from**: it will open a dialog to pick up a language file

- **Generate new (English) and open it**: it will generate a new English language file in location you will choose and open it

- **Generate new (English) only**: it will generate a new English language file in location you will choose without opening it

- **Change font preview**: it will change font for text (translation) entries for a better accessibility

## 4. Language file information

- You can input here language name and code, translator name, translation version, specify language for a WinPaletter version and change layout

![alt text](https://github.com/Abdelrhman-AK/WinPaletter/blob/master/Media/Wiki/LanguageCreation/3.png?raw=true)

## 5. Edit global string

- These are strings that can be translated, mostly used in messages and dialogs (strings written in application's code but not application's designer)

![alt text](https://github.com/Abdelrhman-AK/WinPaletter/blob/master/Media/Wiki/LanguageCreation/4_0.png?raw=true)

- Red color means that text entry is not translated

- You can search here for a text 

![alt text](https://github.com/Abdelrhman-AK/WinPaletter/blob/master/Media/Wiki/LanguageCreation/4_1.png?raw=true)

- You can extend search scope here

![alt text](https://github.com/Abdelrhman-AK/WinPaletter/blob/master/Media/Wiki/LanguageCreation/4_2.png?raw=true)

- You can replace text by this button

![alt text](https://github.com/Abdelrhman-AK/WinPaletter/blob/master/Media/Wiki/LanguageCreation/4_3.png?raw=true)

## 6. Forms strings

- WinPaletter will load instances of all application's forms into memory so that you can modify text and see them properly

![alt text](https://github.com/Abdelrhman-AK/WinPaletter/blob/master/Media/Wiki/LanguageCreation/5_0.png?raw=true)

- When this loading finishes, you can pickup a form you want to modify and open it

![alt text](https://github.com/Abdelrhman-AK/WinPaletter/blob/master/Media/Wiki/LanguageCreation/5_1.png?raw=true)

- This is the appearance of form with all text items (Red colors mean that text entries are not translated)

![alt text](https://github.com/Abdelrhman-AK/WinPaletter/blob/master/Media/Wiki/LanguageCreation/5_2.png?raw=true)

- Click on a text item (rectangle) to edit it in the right sidebar

![alt text](https://github.com/Abdelrhman-AK/WinPaletter/blob/master/Media/Wiki/LanguageCreation/5_3.png?raw=true)

- When you finish item text change, you will see that red color has gone

![alt text](https://github.com/Abdelrhman-AK/WinPaletter/blob/master/Media/Wiki/LanguageCreation/5_4.png?raw=true)

- You can search here for text

![alt text](https://github.com/Abdelrhman-AK/WinPaletter/blob/master/Media/Wiki/LanguageCreation/5_5.png?raw=true)

- You can replace text by this button

![alt text](https://github.com/Abdelrhman-AK/WinPaletter/blob/master/Media/Wiki/LanguageCreation/5_6.png?raw=true)

- Close the child (mini) form when you finish its translation

![alt text](https://github.com/Abdelrhman-AK/WinPaletter/blob/master/Media/Wiki/LanguageCreation/5_7.png?raw=true)

- Repeat previous steps with rest forms

![alt text](https://github.com/Abdelrhman-AK/WinPaletter/blob/master/Media/Wiki/LanguageCreation/5_1.png?raw=true)
