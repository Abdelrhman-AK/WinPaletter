## 1. Disable cursors rendering

- You can use it if you want to keep current cursors and disable WinPaletter from applying new cursors

- Open Settings `>` Theme applying behavior `>` Cursors `>` Uncheck `Automatic apply custom cursors (Creates new scheme called WinPaletter in Control Panel > Mouse)`

![alt text](https://github.com/Abdelrhman-AK/WinPaletter/blob/master/Media/Wiki/Settings/DisableCurRendering.png?raw=true)

***

## 2. Reset cursors to default

- You can use it if you want to reset cursors to default if customizing cursors is disabled in WinPaletter

- Open Settings `>` Theme applying behavior `>` Cursors `>` Check `If Cursors Applying is disabled or skipped, automatic switch the cursors scheme to Windows Default "Aero"`

![alt text](https://github.com/Abdelrhman-AK/WinPaletter/blob/master/Media/Wiki/Settings/CurReset.png?raw=true)
