## 1. Enable/Disable lock toggle

Make it checked to make WinPaletter apply changes to wallpaper

![alt text](Assets//Wallpaper/0.png?raw=true)

***

## 2. `Optional` Toolbar buttons

### 1. Open from WinPaletter theme
This will copy preferences from a WinPaletter theme file into current open theme

### 2. Open from Current applied one
This will copy preferences from current Windows settings into current open theme (looks like an undo)

### 3. Open from Default Windows
This will copy preferences from default Windows theme into current open theme

![alt text](Assets//Wallpaper/1.png?raw=true)

***

## 3. Parts

- You can change wallpaper source, style, wallpaper path, solid color, wallpaper tone and slideshow

![alt text](Assets//Wallpaper/3.png?raw=true)

- Every change you make in these parts will change this preview

![alt text](Assets//Wallpaper/2.png?raw=true)

- Preview loading speed depends on image's size\resolution and your device's performance, so there may be a delay or high memory usage with high resolution images.

***

## 4. Source

- Your wallpaper can be from different sources: picture, tone, color and slideshow

- Wallpaper tone is designed by WinPaletter developer and it is not a Windows feature

- Applying slideshow source requires explorer restart.

![alt text](Assets//Wallpaper/4.png?raw=true)

***

## 5. Style

- You can change wallpaper style. It can be fill, fit, stretch, centered and tile

- Fill and fit positions and slideshow source are not available for Windows XP & Vista.

![alt text](Assets//Wallpaper/5.png?raw=true)

## 10. Actions

1. **Load into current theme:** it will load current wallpaper preferences into current open WinPaletter theme, and will close this form so you can continue modifying other Windows aspects and finally press on Apply on main form to apply all aspects in the open theme.

2. **Quick apply:** it will apply current wallpaper preferences instantly to test their real effects on system.

3. **Cancel:** it will close this form without loading any modification made to wallpaper

![alt text](Assets//Wallpaper/9.png?raw=true)
